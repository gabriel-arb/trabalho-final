Passo 1: Estrutura Básica

Olá, bem vindo ao tutorial de como criar um site, aqui ensinarei como eu fiz esse site que você está usando agora. Comece criando a estrutura básica do seu site com HTML. Utilize tags semânticas para organizar o conteúdo.

          
            <!DOCTYPE html>
            <html lang="pt-br">
            <head>
              <meta charset="UTF-8">
              <meta name="viewport" content="width=device-width, initial-scale=1.0">
              <link rel="stylesheet" href="style.css">
              <title>Meu Site</title>
            </head>
            <body>
              <header>
                <h1>Meu Site</h1>
              </header>
              ...
            </body>
            </html>
